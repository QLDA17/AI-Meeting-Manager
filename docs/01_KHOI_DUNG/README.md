# 🚀 01_KHOI_DUNG - Bắt Đầu Nhanh

Chào mừng đến với MultiMinutes AI! Folder này chứa mọi thứ bạn cần để bắt đầu nhanh chóng.

## 📚 Nội Dung Folder

| File | Mô Tả | Dành Cho |
|------|-------|----------|
| `PROJECT_SUMMARY.md` | Tóm tắt dự án - MultiMinutes AI là gì? | 🆕 Người mới |
| `QUICK_REFERENCE.md` | Lệnh cơ bản & shortcuts | ⚡ Cần chạy nhanh |
| `CLEANUP_LOG.md` | Lịch sử dọn dẹp cấu trúc | 📋 Lịch sử |
| `TRUOC_VA_SAU.md` | So sánh trước/sau cleanup | 📊 So sánh |

## 🚀 Bước Đầu Tiên (5 Phút)

```bash
# 1. Đọc tóm tắt dự án
cat PROJECT_SUMMARY.md

# 2. Xem lệnh cơ bản
cat QUICK_REFERENCE.md

# 3. Chạy dự án
python src/api/main.py        # Backend
cd frontend && npm run dev    # Frontend
```

## 🎯 Cần Gì?

- **Hiểu dự án là gì?** → Bắt đầu từ `PROJECT_SUMMARY.md`
- **Lệnh cơ bản?** → Xem `QUICK_REFERENCE.md`
- **Muốn biết lịch sử?** → Đọc `CLEANUP_LOG.md`
